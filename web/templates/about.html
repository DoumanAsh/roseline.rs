This is a simple front-end to my AGTH database. You can find AGTH hooks, add new ones or edit existing ones. I allow write access to everyone for simplicity sake. In exchange I expect fair use of it from you. When malicious use will be detected I'll shutdown this site.

Adding new hook

You can use VNDB search to find any VN and add new hook for this VN

Finding existing hook

On main page you can search for VNs that has hooks. Note that it expects english titles of VNs. You can also quickly check whether there is any hook by using VNDB's ID: http://roseline.duckdns.org/vn/<vndb_id>

When there is no hook

In most case ITHVNR should handle most VNs. But in cases when it cannot you can try to ask on Hongfire.
